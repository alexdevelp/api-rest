## Rodando Docker ###
** docker --help **
1) criando container
->  docker run --name postgres -e POSTGRES_PASSWORD=docker -p 5433:5432 -d postgres

2) rodando imagem
-> docker start <nome_image>



## TypeOrm ##
# comando para criar a migration, com flag para informar o nome da migration
-> yarn typeorm migration:create -n CreateProducts

# comando para rodar a migration que criamos
-> yarn typeorm migration:run

# comando de migration para adicionar uma coluna
-> yarn typeorm migration:create -n AddCustomerIdToOrders

# comando para acicionar uma chave estrangeira


# ordem de criação - arquiterura
1- migration (create | run)
2- modules
3- entities
4- repositories
5- services
6- controllers
7- routes

## Redis - Cache ##
-> docker exec -it redis sh
-> redis-cli
-> set teste testando
-> get teste
-> del "para deletar dado, informar chave ou array de chave"

# comando para criar uma aplicação web para consulta do Cache, vai criar no localhost da maquina: http://localhost:8001
-> docker run --name redis-client -v redisinsight:/db -p 8001:8001 -d -t redislabs/redisinsight:latest
